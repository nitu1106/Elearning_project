# EduSphere — Complete E-Learning Platform

## Folders
- `elearning-backend/`  — Node.js + Express + MongoDB REST API
- `elearning-frontend/` — React 18 SPA

## Quick Start
1. `cd elearning-backend && npm install` → fill `.env` → `npm run dev`
2. `cd elearning-frontend && npm install` → `npm start`
3. Open http://localhost:3000

See `elearning-frontend/README.md` for full setup guide.
